## Differences

- Fastest build with esbuild and tuned babel
- New esnext target
- Backwards compatible

## Installation

We recommend to use [yarn](https://classic.yarnpkg.com/en/docs/install/) for dependency management:

```shell
yarn add @mntm/vkui-builder
```
